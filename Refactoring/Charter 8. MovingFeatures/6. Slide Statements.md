#### Slide Statements
Известен так же как: Consolidate Duplicate Conditional Fragments

### До
```cs
void Process(int x)
{
    if (x > 0)
    {
        Console.WriteLine("Positive");
        DoSomething();
    }
    else if (x < 0)
    {
        Console.WriteLine("Negative");
        DoSomething();
    }
    else
    {
        Console.WriteLine("Zero");
        DoSomething();
    }
}

void DoSomething() { /* some code */ }
```
### После
```cs
void Process(int x)

    if (x > 0)
    {

    }

    {

    }

    {

    }

    DoSomething();
```
