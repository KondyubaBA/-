### 3. Rename Method (Переименование метода)
Изменяет имя метода для более точного отражения его функции.

#### До
```cs
public List<User> GetUserDetails()
{
    // код для получения списка пользователей
}
```

#### После
```cs
public List<User> GetUsers()
{
    // код для получения списка пользователей
}
```
